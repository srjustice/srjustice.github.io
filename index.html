At Liberty, I worked alongside Senior Quality Analysts to iteratively test Liberty’s internally-developed web-related products and help rebuild them when bugs and errors were found.

I have been consistently commended by professors and job supervisors for my programming/problem-solving abilities; strong grasp of technologies; attention to detail; and ability to research, learn, and overcome obstacles independently.

I offer every project the best of my time, energy, and knowledge in order to achieve the project's goals with excellence.

Work Experience

May - August, 2016

Air Force Institute of Technology
Software Engineering Intern

Description:

My internship at AFIT consisted of providing research and support for ASYS 632, AFIT's Satellite Design and Test course. Throughout ASYS 632, students design, build, and test a CubeSat (U-class spacecraft). My primary responsibilities included programming and integrating the embedded computing devices that comprise the CubeSat, writing software for the MATLAB Ground Station used to command the CubeSat, and developing/giving lectures to the ASYS 632 students.

Notable Accomplishments:

Increased the security of transmissions sent between the CubeSat and the Ground Station by writing code that implements cyclic redundancy checks (CRCs) and authentication counts in transmissions. Analyzed the efficiency of memory usage on the CubeSat and was able to reduce the CubeSat’s memory usage by over 50%. Utilized FreeRTOS to write the operating system for a brand new microcontroller-based circuit board AFIT developed. Successfully integrated the new circuit board into the CubeSat such that it was able to communicate with the rest of the CubeSat bus and be commanded from the Ground Station. Developed code for the CubeSat that gave it the ability to respond to commands with NACKs (negative-acknowledgments), and developed code for the Ground Station that allows it to parse and display NACKs to Ground Station users. Developed a new iterative method for transferring files from the CubeSat to the Ground Station that is faster and more efficient than the previous method. Helped develop a new MATLAB GUI that parses telemetry log files and presents the historic telemetry in an informative manner. Developed code for the Ground Station that allows it to display descriptions and default values for command parameters, which is helpful for new Ground Station users. Developed and presented four lectures/labs to the ASYS 632 graduate students: "Introduction to Atmel Studio 6.2", "Introduction to C Programming", "C Programming Lab", and "Ground Station Demo".
